Today's Lesson

Hacking! Coming into this class I didn't think that a thing like HTML or Javascript could be for hacking in ethical or unethical ways.
I found it rather interesting how using HTML you could inspect a web pages elements through the browser and alter the code during a
purposeful failed login to crack a low level secured website. This is possible by removing the failed login Javascript code and replacing it with a valid
line instead. This method is again only for simple protected websites, however there is a great need for precise detail to ensure that the
process is done right.

Another type of hacking I learned of is called "ethical hacking". This is where people try to find flaws and gaps that can be fixed before someone
with bad intentions try to gain access inside the network. Javascript is used inside of html to try to find ways into servers because this can escalate
into gaining access to networks and endless possibilities for the hacker to exploit. Another form of hacking is for personal use, here the goal is just
to achieve a more effecient browser and a better experience on the web. One can create add-ons with javascript or insert simple lines of code to learn about
the document's details, removing ads and pictures, and other things to better personl use of the web.
